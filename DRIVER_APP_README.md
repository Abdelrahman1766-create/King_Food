# تطبيق سائق KING FOOD

تطبيق منفصل للسائقين في نظام توصيل الطعام KING FOOD.

## الميزات

- **تسجيل الدخول**: مصادقة آمنة للسائقين
- **عرض الطلبات**: رؤية الطلبات المخصصة للسائق فقط
- **تحديث حالة الطلب**: تحديث حالة الطلبات من "في الطريق" إلى "تم التوصيل"
- **تفاصيل الطلب**: عرض تفاصيل كاملة للطلبات
- **خريطة التنقل**: (قريبًا) خرائط للتنقل إلى عنوان العميل
- **الاتصال بالعميل**: (قريبًا) إمكانية الاتصال بالعميل

## متطلبات التشغيل

- Flutter SDK 3.0+
- Firebase project مع تفعيل Authentication و Firestore
- حساب سائق مسجل في نظام الإدارة

## إعداد المشروع

1. **إعداد Firebase**:
   ```bash
   # إضافة Firebase إلى المشروع
   flutter pub add firebase_core firebase_auth cloud_firestore
   ```

2. **تكوين Firebase**:
   - أنشئ مشروع Firebase جديد أو استخدم المشروع الموجود
   - فعل Authentication مع Email/Password
   - فعل Firestore Database
   - أضف تطبيق Flutter و قم بتحميل ملفات التكوين

3. **تشغيل التطبيق**:
   ```bash
   # تشغيل تطبيق السائق
   flutter run lib/main_driver.dart
   ```
