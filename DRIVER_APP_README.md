# تطبيق سائق KING FOOD

تطبيق منفصل للسائقين في نظام توصيل الطعام KING FOOD.

## الميزات

- **تسجيل الدخول**: مصادقة آمنة للسائقين مع Firebase
- **عرض الطلبات**: رؤية الطلبات المخصصة للسائق فقط
- **تحديث حالة الطلب**: تحديث حالة الطلبات من "في الطريق" إلى "تم التوصيل"
- **تفاصيل الطلب**: عرض تفاصيل كاملة للطلبات مع الأسعار والعناوين
- **خريطة التنقل**: ✅ خرائط Yandex Maps للتنقل إلى عنوان العميل مع عرض نقطة الاستلام والتسليم
- **الاتصال بالعميل**: ✅ إمكانية الاتصال بالعميل مباشرة من التطبيق
- **تتبع الموقع**: ✅ تتبع الموقع الجغرافي الحالي للسائق

## متطلبات التشغيل

- Flutter SDK 3.0+
- Firebase project مع تفعيل Authentication و Firestore
- حساب سائق مسجل في نظام الإدارة

## إعداد المشروع

1. **إعداد Firebase**:
   ```bash
   # إضافة Firebase إلى المشروع
   flutter pub add firebase_core firebase_auth cloud_firestore
   ```

2. **تكوين Firebase**:
   - أنشئ مشروع Firebase جديد أو استخدم المشروع الموجود
   - فعل Authentication مع Email/Password
   - فعل Firestore Database
   - أضف تطبيق Flutter و قم بتحميل ملفات التكوين

3. **تشغيل التطبيق**:
   ```bash
   # تشغيل تطبيق السائق
   flutter run lib/main_driver.dart
   ```

## استخدام التطبيق

### تسجيل الدخول
- استخدم البريد الإلكتروني وكلمة المرور المسجلة في نظام الإدارة
- السائقون يتم إنشاؤهم من قبل الإدارة

### إدارة الطلبات
- عرض الطلبات المخصصة للسائق فقط
- تحديث حالة الطلب من "في الطريق" إلى "تم التوصيل"
- عرض تفاصيل الطلب الكاملة

### الخريطة والتنقل
- رؤية خريطة Yandex Maps مع مواقع الاستلام والتسليم
- الخط الأخضر يوضح المسار بين النقطتين
- يمكن التكبير والتصغير والتنقل على الخريطة

### الاتصال بالعميل
- الضغط على زر الهاتف للاتصال المباشر بالعميل
- يتم استخدام تطبيق الهاتف الافتراضي للجهاز
