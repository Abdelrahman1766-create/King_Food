# 🇷🇺 خيارات الدفع المتاحة في روسيا

## المشكلة:
Stripe غير متاح في روسيا بسبب القيود الجيوسياسية والعقوبات الاقتصادية.

## الحلول المتاحة:

### 1️⃣ **YooKassa** (الخيار الأول والأفضل) ⭐⭐⭐⭐⭐
```
📊 الحالة: ✅ متاح بالكامل في روسيا
🌐 الموقع: https://yookassa.ru
💳 طرق الدفع:
   ✓ بطاقات Visa/Mastercard
   ✓ Mir (البطاقة الروسية الوطنية)
   ✓ Apple Pay و Google Pay
   ✓ Sberbank Online
   ✓ Yandex.Kassa
   ✓ محافظ رقمية أخرى
💰 العمولة: من 1.5% إلى 2.5%
🔒 الأمان: مستوى عالي جداً (PCI DSS)
📚 التوثيق: ممتازة ومفصلة جداً
🚀 الأداء: سريعة جداً
```

**الخطوات:**
1. اذهب إلى: https://yookassa.ru
2. انقر "Для ИП и ООО" (للمتاجر)
3. أو "Для физических лиц" (للأفراد)
4. ملء البيانات والتحقق
5. احصل على Shop ID و Secret Key

## خطوات البدء السريعة:

### الخطوة 1: التسجيل
```
👉 اذهب إلى: https://yookassa.ru
👉 اضغط "Зарегистрироваться" (Sign up)
👉 ملء البيانات الأساسية
```

### الخطوة 2: التحقق
```
👉 قم بتحميل وثائق التحقق (جواز سفر، وثائق العمل)
👉 انتظر الموافقة (عادة 1-2 يوم)
```

### الخطوة 3: الحصول على المفاتيح
```
👉 دخول لوحة التحكم (Dashboard)
👉 الذهاب إلى: Настройки → API ключи
👉 انسخ:
   - Shop ID (معرف المتجر)
   - Secret Key (المفتاح السري)
```

### الخطوة 4: التطبيق
```dart
static const String yooKassaShopId = 'YOUR_SHOP_ID_HERE';
static const String yooKassaSecretKey = 'YOUR_SECRET_KEY_HERE';
```

### الخطوة 5: الاختبار
```
بطاقة الاختبار:
- الرقم: 4111 1111 1111 1111
- CVC: أي 3 أرقام
- الصلاحية: أي تاريخ مستقبلي
```
